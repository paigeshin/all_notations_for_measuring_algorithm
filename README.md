# all_notations_for_measuring_algorithm

**o(n^2), Small Case o** 

⇒ Faster than n^2 

**O(nlogn), Large Case O**

⇒ At worst, will be nlogn

**Θ(n), Theta**

⇒ Equal to n 

**Ω(n^2), Large Case Omega**

⇒ Slower or equal to n^2

⇒ At best, will be n^2

**ω(n^3), Small Case Omega** 

⇒ Slower than n^3
